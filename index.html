• Tower of hanoi

About

Tower of Hanoi is a game where there are three piles and you need to move all the blocks from one pile to another pile.


But there is a rule, during the game you cannot place a bigger block on top of a smaller one.


This software is an animation from the solution for an specified number of blocks.


**This software uses a recursive solution. To avoid excessive memory consumption, the value of the number of blocks is limited to fifteen.**

Please use your device in the landscape mode ;)
